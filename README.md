## ✨ Características

- Dark mode premium con fondo radial
- Layout grid responsivo 3 columnas (mobile-first)
- Animaciones CSS (pulse, ping, shimmer)
- Avatar DiceBear · Badge de disponibilidad · Widget de ubicación
- Terminal / IDE estilo macOS con filosofía de trabajo
- Botones: GitHub, LinkedIn, CodePen, Portafolio, WhatsApp, Email

## 🛠️ Stack

| Capa | Tecnología |
|---|---|
| Framework | React 18 + Vite 6 |

## 🚀 Desarrollo

```bash
npm install
npm run dev      # http://localhost:5173/
npm run build    # Producción en /dist
```
